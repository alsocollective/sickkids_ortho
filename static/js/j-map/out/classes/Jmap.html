Jmap Class

Module: Jmap

Jmap is the container for all the action going on with this plugin for Jquery

Created by:

Bohdan Anderson @ Also Collective

Constructor

Jmap

(
  • [options]
)

Defined in j-map.js:14

Parameters:

  • [options] Object optional

    Data that specifies, the functionality and look of the plugin

    • [apiKey] String optional

      Needed for this plugin to work, go to google to get the api key

    • [locationLat] Number optional

      Latitude of map on load

    • [locaiotnLon] Number optional

      Longitude of map on load

    • [autoStart] Boolean optional

      If 'true', will start loading the map right way, 'false', will wait

    • [mapStyles] String optional

      Generate the Json format from here

    • [defaultZoom] Number optional

      Default is 8, determines the zoom on load

    • [oneIconAtATime] Boolean optional

      Default is true, will hide all other info bubbles when opening a new one

    • [funOnClick] Funciton optional

      this is ran when the marker is clicked

    • [funOffClick] Funciton optional

      this is ran when an other marker is clicked

    • [infoPadding] Number optional

      px size

    • [infoArrowSize] Number optional

      px size

    • [infoArrowPosition] Number optional

      from bottom left px amount

    • [infoArrowStyle] Number optional
    • [infoBorderRadius] Number optional

      px size

    • [infoBorderWidth] Number optional

      px size

    • [infoBorderColor] Color optional

      rgb or hex or w.e colour means you want

    • [infoShadowStyle] Number optional

      0,1,2

    • [infoBackgroundColor] Color optional

      rgb or hex or w.e colour means you want

    • [infoBackgroundClassName] String optional

      string value, default is gmap-overlay

    • [infoHideCloseButton] Boolean optional

      Hide button...

    • [infoDisableAnimation] Boolean optional

      Pop in animation

    • [infoDisableAutoPan] Boolean optional

      Centers the map to location of marker when clicked

addMarkerType

(
  • options
)

Defined in j-map.js:186

Generates a new marker,
reference the created markers in order of creation starting with 0.
You can use origin on a sprite sheet to use one image for all your icons

Parameters:

  • options Object
    • url String

      Url of the icon to be used

    • size Array

      Size of the icon to be loaded, [100,100]

    • origin Array

      Origin, what corner should be thought of as zero, [0,0]

    • anchor Array

      Anchor, the moving location... I use this to zero, [20,90]

    • scaledSize Array

      ScaleSize, what size the icon should be shown as

Example:

myMap.addMarkerType({
    url: '/static/img/takeaction-btn.png',
    size: [100,100],
    origin: [0,0],
    anchor: [20,90],
    scaledSize:[100,100]
})

loadData

(
  • json
)

Defined in j-map.js:305

Loads Marker data into map.
Be sure to create the icon types before loading.

Parameters:

  • json Object

    location data

    • name String

      What will show up for mouse over, not really special

    • lat Number

      Latitude of marker

    • lon Number

      Longitude of marker

    • icon Number

      The numbers here relate to the order in which .addMarkerType are created

    • altIcon Number

      when clicked this Icon will be shown

    • description String

      The text will be displayed in the text box

Example:

{"data":[
    {
        "name":"Us",
        "lat":43.649517,
        "lon":-79.397373,
        "icon": 0,
        "altIcon":1,
        "description":"some test info"
    },
    {
        "name":"Them",
        "lat":43.648717,
        "lon":-79.396475,
        "icon": 0,
        "altIcon":1,
        "description":"some test info"
    },

]}

Makes the map cluster elements,
for more reference look at link

Parameters:

  • cOptions Object
    • imagePath String

      Url of the icon to be used, this is a bit funny, for we don't need to end the string with .png and 1 must be the last letter in the name of the file

    • imageSizes Array

      Size of the icon to be loaded, [100,100]

    imageSizes: [100,100],

});
